Golf Scorecard App

Technologies: MongoDB, Node.js, React Native, Postman

I developed a robust golf tracking app that allows users to input their scores and monitor their performance over time. The app features a MongoDB backend for persistent data storage, ensuring that users can reliably save and retrieve their golf statistics. On the server-side, I used Node.js to handle the application logic, while React Native was employed to create a responsive and intuitive mobile interface. This seamless combination of technologies provides a smooth user experience across various devices, enabling golfers to keep track of their progress anytime, anywhere.

Yap Messaging App

Technologies: React Node.js, Express

Collaborating with a team of friends, I played a key role in developing Yap, a dynamic messaging application. My contributions focused on designing and implementing the front-end login page, ensuring it was both user-friendly and visually appealing. I utilized React to build the interactive interface, while the backend was powered by Node.js and Express. This integration allowed for efficient user authentication and management, providing a secure and streamlined user experience. Yap is a testament to my ability to work effectively in a team and deliver a polished product.

Web Proxy Server

Technologies: C

I implemented a Web Proxy Server capable of accepting client requests and converting them to HTTPS for secure communication with web servers. This project involved in-depth understanding of network protocols and secure data transmission, providing a crucial learning experience in system-level programming with C. The server ensures secure and efficient data exchange, highlighting my skills in network security and low-level programming.

Simple IPv4 Router and Network Architecture with VLANs

Technologies: Python, Cisco Packet Tracer

In this project, I developed a multi-subnet network topology for a simulated company environment. Using Mininet, I integrated switches and routers to create a network architecture that segmented different departments using VLANs (Virtual Local Area Networks). This setup ensured network isolation and security, reflecting my proficiency in network architecture and Python scripting. The project demonstrates my ability to design and implement complex networking solutions tailored to specific organizational needs.

UDP File Echo Application

Technologies: C

I developed a client-server file echo application using the UDP protocol in a UNIX environment. This application featured a custom protocol for splitting files into MTU-sized packets, ensuring reliable file transfer over the network. The project enhanced my understanding of network protocols, particularly UDP, and showcased my skills in handling low-level data transmission tasks in C.
